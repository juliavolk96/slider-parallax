# Dynamic Parallax Slider

A web interface for browsing anime titles, featuring real-time data integration and advanced CSS parallax effects.

## Live Demo
You can view the live project here:
[https://juliavolk96.github.io/slider-parallax/](https://juliavolk96.github.io/slider-parallax/)

## Tech Stack
* **Frontend**: HTML, CSS, JavaScript (ES6+)
* **Libraries**: Swiper.js
* **API**: Jikan REST API

## Project Structure
* `index.html` — Semantic structure and slide content.
* `css/style.css` — Custom styles and animation logic.
* `js/script.js` — Slider initialization and asynchronous API handling.
